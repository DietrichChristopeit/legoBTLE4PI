legoBTLE.user.Experiment.Experiment¶

class legoBTLE.user.Experiment.Experiment(name, loop, measure_time=False, debug=False)¶

Bases: object

This class models an Experiment

This class models an Experiment that can be performed with the LEGO|copy| devices (Motors etc.). It is suggested to use this class to create and run_each sequences of commands concurrently.

However, the class is mainly a wrapper with some convenience functions. Nothing stands against using the ‘lower level’ functions.

Parameters

  • name (str) – A descriptive name.

  • measure_time (bool) – If set, the execution time to process the Action List will be measured.

  • debug (bool) – If set, function call info is printed.
