legoBTLE.device.SynchronizedMotor.SynchronizedMotor¶

class legoBTLE.device.SynchronizedMotor.SynchronizedMotor(motor_a, motor_b, server, name='SynchronizedMotor', time_to_stalled=None, stall_bias=0.2, debug=False)¶

Bases: legoBTLE.device.AMotor.AMotor

This is the Synchronized Motor.

The synchronized Motor consists of 2 SingleMotors plugged together to form one.

This class models the user view of two motors chained together on a common port.

The available commands are executed in synchronized manner, so that the motors run_each in parallel and at least start at the same point in time. See also the LEGO Wireless Protocol 3.0.00r.17.
